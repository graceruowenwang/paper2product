# 首图生成 Prompt 集合

> 用于 Midjourney / Grok Imagine / DALL-E 生成文章首图。

---

## 通用基础 Prompt

```
A minimalist tech illustration showing [论文主题相关概念],
clean geometric shapes, blue-purple gradient background (#1A56DB to #7C3AED),
white lines and dots forming network connections,
modern data visualization style,
no text, no people, 16:9 aspect ratio, high resolution
```

---

## 按论文方向的变体

### NLP / 对话系统

```
A minimalist illustration of conversation bubbles transforming
into structured data, neural network nodes connecting them,
blue-purple gradient background, clean geometric design,
abstract tech style, no text, no people, 16:9
```

### 推荐系统

```
Abstract visualization of recommendation connections,
user icons connecting to items through glowing lines,
bipartite graph style, blue-purple color scheme,
minimalist tech illustration, no text, 16:9
```

### 计算机视觉

```
Abstract eye or camera lens with digital grid overlay,
image fragments being analyzed by geometric patterns,
blue-purple gradient background, futuristic clean design,
no text, no people, 16:9
```

### AI Agent / 自主系统

```
Abstract autonomous agent navigating a digital landscape,

```

### 多模态

```
Multiple data streams (text, image, audio) converging

```

### 通用 RAG / 检索增强


---

## 风格注意事项

1. **一致性**：所有首图保持相同的色调和风格
2. **简洁**：一张图只表达一个核心概念
3. **品牌色**：始终使用蓝紫渐变作为主色调
4. **无文字**：文字用排版工具后期添加
5. **抽象**：用几何图形和线条表达概念，不要写实

---

## 后期处理

生成图片后，用 Figma / Canva 添加：
- 右下角 "Paper2Product" 水印
- 可选：左上角论文方向标签（如 "NLP" "推荐" "Agent"）
- 确保输出尺寸 900 x 383 像素
