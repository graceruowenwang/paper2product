# 公众号配置指南

---

## 菜单栏设计

```
底部菜单（3 个一级菜单）：

| 论文解读 | 工具箱 | 关于 |
|----------|--------|------|
| 本周精选  | AI 写作 | 关于我们 |
| 往期目录  | 选题投票 | 加入社群 |
| 搜索论文  | 免费资源 | 商务合作 |
```

### 各菜单说明

**论文解读**
- 本周精选：跳转到最新周报
- 往期目录：按分类展示历史文章（外链到文章合集页）
- 搜索论文：跳转到搜索页面或 Notion 数据库

**工具箱**
- AI 写作：推荐 AI 辅助写作工具/教程
- 选题投票：读者投票选下周想看的论文
- 免费资源：推荐的 Prompt、工具列表

**关于**
- 关于我们：Paper2Product 介绍和定位
- 加入社群：微信群二维码或入群方式
- 商务合作：联系方式

## 自动回复设置

### 关注后自动回复

```
欢迎关注 Paper2Product！

我是 AI 产品经理视角的论文解读站。
每周帮你从 arXiv 海量论文中，筛选出最具产品价值的前沿研究。

回复数字获取内容：
1 - 本周最值得读的论文
2 - 最热门文章 TOP 5
3 - AI 产品经理必备工具包

回复关键词获取专题：
「RAG」- 检索增强相关解读
「Agent」- AI Agent 相关解读
「推荐」- 推荐系统相关解读
```

### 关键词自动回复

| 关键词 | 回复内容 |
|--------|---------|
| 1 | 最新周报链接 |
| 2 | 热门文章列表 |
| 3 | 工具包文章链接 |
| RAG | RAG 专题文章列表 |
| Agent | Agent 专题文章列表 |
| 推荐 | 推荐系统专题文章列表 |
| 投稿 | 投稿/建议邮箱或表单 |
| 合作 | 商务合作联系方式 |

### 收到消息默认回复

```
感谢你的留言！

如果你有想了解的论文方向，可以回复论文标题或关键词。
我们会优先解读呼声最高的论文。

需要人工服务，请回复「合作」。
```
