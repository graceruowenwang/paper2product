# Prompt: 生成摘要与导语

> 用于生成文章开头的摘要/导语，吸引读者继续阅读。

---

## 微信摘要（用于分享卡片）

```
请为以下文章生成一个微信分享摘要（显示在分享卡片上）。

要求：
- 60 字以内
- 包含核心产品价值点
- 激发阅读兴趣
- 不透露太多细节

文章内容：[一句话总结]
```

---

## 文章导语（开篇段落）

```
请为以下论文解读文章撰写开篇导语（2-3 段）。

论文：[标题]
核心洞察：[一句话产品价值]
目标读者：AI 产品经理

导语要求：
1. 第一段：用一个场景或问题引发共鸣（读者能代入的痛点）
2. 第二段：引出论文的核心能力（用一句话说清它能解决什么）
3. 第三段：预告文章的产品化分析（告诉读者读完能得到什么）

风格：
- 像跟同事聊天，不要端着
- 避免用"本文将..."这种论文式开头
- 可以用"你"来增加对话感
- 总长度 100-150 字
```

---

## 周报导语

```
请为本周的 AI 论文周报写一段导语。

本周关键趋势：[描述]
Top 论文方向：[描述]

要求：
- 2-3 句话
- 总结本周最重要的趋势变化
- 用产品经理的视角
- 引导读者关注本周最有价值的论文
```
