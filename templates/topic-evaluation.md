## 评估维度（每项 1-5 分）

### A. 产品相关性（权重 x2）

这篇论文解决的问题是否对应真实的产品需求？

- [ ] 5分：直接解决一个已验证的市场痛点
- [ ] 4分：能显著改善现有产品的体验
- [ ] 3分：有潜在应用场景，但需要验证
- [ ] 2分：技术有趣但应用场景模糊
- [ ] 1分：纯学术研究，暂无产品价值

**评分：___**

### B. 时效性（权重 x1.5）

- [ ] 5分：刚发布（<1周），行业高度关注
- [ ] 4分：近期发布（1-2周），与当前热点相关
- [ ] 3分：发布时间适中，但话题持续热门
- [ ] 2分：有一定时效价值
- [ ] 1分：时效性不强

**评分：___**

### C. 读者覆盖面（权重 x1.5）

- [ ] 5分：几乎所有 AI PM 都会关心
- [ ] 4分：特定领域 PM（如推荐、搜索）高度关注
- [ ] 3分：对部分 PM 有参考价值
- [ ] 2分：小众领域
- [ ] 1分：非常窄的专业方向

**评分：___**

### D. 技术可解释性（权重 x1）

能否用产品经理能理解的方式解释清楚？

- [ ] 5分：概念直觉上好理解，类比容易找
- [ ] 4分：需要一些背景知识但可以解释
- [ ] 3分：有一定门槛但值得讲
- [ ] 2分：需要较深技术背景
- [ ] 1分：很难通俗化

**评分：___**

### E. 可执行性（权重 x1）

论文成果是否容易转化为实际产品？

- [ ] 5分：有开源代码，可直接使用
- [ ] 4分：有清晰的实现路径
- [ ] 3分：需要一定的工程努力
- [ ] 2分：实现成本较高
- [ ] 1分：还停留在理论阶段

**评分：___**

## 决策

- **8-10 分**：优先选题，本周安排写作
- **7-8 分**：纳入备选，视时间安排
- **5-7 分**：记录在案，可能用于周报速览
- **< 5 分**：放弃
