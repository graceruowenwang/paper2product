# Paper2Product 周报模板

> 每周五发布，回顾本周最值得关注的 AI 论文动态。

---

## 标题格式

`P2P 周报 #XX | 本周 N 篇论文值得你关注`

---

## 正文结构

### 1. 本周一句话

用一句产品经理视角的话总结本周 AI 领域最重要的趋势变化。

### 2. 本周 Top 论文（2-3 篇）

每篇用以下格式，每篇控制在 150-200 字：

**论文标题**
- 一句话价值：
- 核心突破：
- 产品机会：
- 推荐阅读：[全文解读](链接) / 原文链接

### 3. 值得关注的论文速览（5-8 篇）

表格形式：

| 论文 | 来源 | 一句话价值 | 产品化潜力 |
|------|------|-----------|-----------|
| | | | |

### 4. 本周行业动态

- 大厂发布/产品更新（1-2 条）
- 开源项目动态（1-2 条）
- 融资/商业新闻（1-2 条）

### 5. 下周关注

预告下周值得期待的论文发布/会议/事件。

---

## 写作要求

- 全文 1500-2000 字
- 保持信息密度，不说废话
- Top 论文部分必须有产品视角的分析
- 速览表格是最核心价值，认真筛选
