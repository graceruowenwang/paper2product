# 排版与发布 Checklist

> 每篇文章发布前必须过此清单，防止低级错误。

---

## 内容检查

- [ ] 标题是否突出产品价值（不是学术概念）
- [ ] 开篇一句话价值是否足够吸引人
- [ ] 全文无数学公式（用文字解释直觉）
- [ ] 产品化分析占全文 40% 以上
- [ ] 每个产品建议都是具体可执行的
- [ ] 技术概念都有类比解释
- [ ] 文章字数在 2500-4000 字范围
- [ ] 无错别字、无语法错误

## 排版检查

- [ ] 小标题层级清晰（不超过 3 级）
- [ ] 关键信息加粗
- [ ] 表格格式正确
- [ ] 图片/配图已上传且清晰
- [ ] 首图风格统一（使用品牌模板）
- [ ] 代码块格式正确（如有）
- [ ] 外部链接可点击
- [ ] 论文链接和代码仓库链接正确

## SEO 与传播

- [ ] 摘要（导语）包含关键词
- [ ] 文末有引导关注的话术
- [ ] 添加了合适的标签
- [ ] 文末有论文信息卡
- [ ] 分享卡片标题和描述已设置

## 发布后动作

- [ ] 发布后 1 小时内检查排版效果
- [ ] 在相关社群/朋友圈分享
- [ ] 回复前几条留言
- [ ] 记录文章数据（阅读、分享、新增关注）
- [ ] 将文章文件移到 papers/done/

## 发布时间建议

- 周二、周四上午 8:00-9:00（深度解读）
- 周五下午 17:00-18:00（周报）
- 避开：周末、节假日、晚上 10 点后
